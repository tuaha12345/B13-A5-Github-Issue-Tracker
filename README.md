1️⃣ What is the difference between var, let, and const?
=>Var is old way to declare variable and can be re-declare and update where let is new way that can update but not 
  re-declare in the same scope and const used for values that should not change and cannot be updated or re-declared.

2️⃣ What is the spread operator (...)?
=> This is used to copies/expand the old object/array and adds new data.

3️⃣ What is the difference between map(), filter(), and forEach()?
=> Map creates a new array after modifying each element where filter creates a new array with elements that match a condition and forEach is actually works like a loop but doesn't return a new array

4️⃣ What is an arrow function?
=> To make code short and clean this function is being used. Actually, this is a short way to write a function.

5️⃣ What are template literals?
=> ` ` -- this is used for template literals and template literals are used to insert variables inside strings easily.
